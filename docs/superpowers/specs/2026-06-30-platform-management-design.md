# プラットフォーム別管理 設計

作成日: 2026-06-30

## 目的

積みゲートラッカーで、各ゲームを「どのプラットフォームのソフトか」で管理できるようにする。
入力・カード表示・絞り込みの3点を追加する。

## 決定事項

| 論点                        | 決定                                                         |
| --------------------------- | ------------------------------------------------------------ |
| 1ゲームのプラットフォーム数 | 1つだけ                                                      |
| 選び方                      | 固定リストから選ぶ（`<select>`）                             |
| リスト内容                  | Switch / PS5 / PS4 / Steam / PC / スマホ / その他            |
| フィルター                  | 付ける（プラットフォーム別の絞り込み）                       |
| 未設定の扱い                | NULL を「未設定」として扱う。既存データは触らない            |
| フィルターUI                | プラットフォームはドロップダウン（ステータスはボタンのまま） |

## データモデル（games テーブル）

- 追加カラム: `platform`（text・NULL 許容）
- 保存する値: 表示文字列をそのまま保存（例: `"Switch"`, `"PS5"`）
  - キー（`"switch"`）には変換しない。リストが固定かつ日本語なので変換不要でシンプル。
- NULL = 「未設定」。既存レコードは NULL のまま残す。

固定リストの定義（配列として1か所に持つ）:

```js
const PLATFORMS = ["Switch", "PS5", "PS4", "Steam", "PC", "スマホ", "その他"];
```

## UI 設計

### 入力（追加フォーム）

- タイトル入力の隣に `<select>` を追加。
- 選択肢: 先頭に「未設定」（value=""）＋ `PLATFORMS` を `.map()` で描画。
- 初期値は空文字（＝未設定）。
- `useState` で `platform` を管理し、`addGame` の insert に含める。
  - 未設定（空文字）のときは `null` として保存する（NULL に統一）。

### カード（GameCard）

- ステータスバッジの隣にプラットフォームのバッジを1個追加。
- 値があればその文字列、NULL/空なら「未設定」をグレーで表示。

### フィルター（GameListPage）

- プラットフォーム用の `<select>` ドロップダウンを追加。
- 選択肢: 「すべて」（value="all"）＋ `PLATFORMS` ＋「未設定」。
- ステータスフィルター（ボタン）はそのまま残す。
- 絞り込みは **ステータス AND プラットフォーム** の合わせ技。
  - `visibleGames` の算出を、status 条件と platform 条件の両方を満たす形に拡張する。
  - 「未設定」選択時は `platform == null`（または空）のものだけ表示。

## 触るファイル

| ファイル                        | 変更                                                                       |
| ------------------------------- | -------------------------------------------------------------------------- |
| Supabase（SQL・ダッシュボード） | `games` に `platform` カラム追加。IK が実行                                |
| src/pages/GameListPage.jsx      | platform state 追加・insert 修正・filter 拡張・入力/フィルター select 配置 |
| src/components/GameCard.jsx     | プラットフォームバッジを1個追加                                            |

## スコープ外（今回やらない）

- 1ゲーム複数プラットフォーム
- プラットフォームリストの編集 UI
- 既存データへの一括プラットフォーム設定

## 完了の定義

- 追加フォームでプラットフォームを選んでゲードを登録できる。
- カードに選んだプラットフォーム（または「未設定」）が表示される。
- ドロップダウンでプラットフォーム別に絞り込め、ステータスフィルターと併用できる。
- 本番（Vercel）に反映され、既存データが壊れていない。
